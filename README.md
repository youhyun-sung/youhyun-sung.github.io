# youhyunsung.github.io

Personal academic-style website for **Youhyun Sung**, built with Jekyll using a Minimal Mistakes-based template (forked structure of [academicpages](https://academicpages.github.io/)).

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000`.

## Structure

- `_config.yml` — site configuration
- `_pages/` — About, CV, Experience, Contact pages
- `_data/` — navigation and author info
- `_layouts/`, `_includes/`, `_sass/` — theme
- `assets/`, `images/` — static assets
